# authentication-log-analyzer
A Python-based security tool that analyzes authentication logs and detects suspicious login behavior such as repeated failed login attempts.

## Features
- Detects repeated failed login attempts
- Identifies suspicious IP addresses
- Generates a simple security alert report
- Demonstrates basic security monitoring concepts

## Technologies
- Python
- Regex
- Log file analysis

## Example Output
Suspicious Login Activity Report

Suspicious activity detected from IP: 192.168.1.10
Failed attempts: 3

## Use Case
This project demonstrates how security teams can analyze authentication logs to detect potential brute-force attacks or unauthorized login attempts.
